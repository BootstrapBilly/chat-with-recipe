# Features

## Core Experience
- Upload a recipe (PDF/TXT) and instantly start a guided cooking session.
- Chat with the assistant for help, substitutions, and step guidance.
- Track cooking progress with a clear step-by-step flow.
- Scale servings with a dedicated control on both mobile and desktop.
- Swap ingredients from the ingredients panel with immediate UI updates.
- Mobile and desktop layouts tuned for their screen sizes.
- Friendly error handling for failed uploads.
